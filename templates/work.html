001 — Python Password Checker

This Python project uses the Have I Been Pwned API (pwnedpasswords.com) to check whether a given password has appeared in any known data breaches. It's a practical security tool that demonstrates both Python programming skills and an understanding of real-world API integration.

What makes this project particularly interesting from a security standpoint is the use of k-anonymity — a privacy-preserving technique that ensures your actual password is never transmitted over the internet. Only the first 5 characters of the password's SHA-1 hash are sent to the API, and the matching is done locally.

"password is never sent/passed through the internet for checking" — github.com/ViliTajnic/passwordchecker

How It Works

The script takes a password as input, generates a SHA-1 hash, sends the first 5 characters to the HIBP API, and receives back a list of hash suffixes with breach counts. It then checks locally if the full hash appears in the response — ensuring complete password privacy throughout the entire process.

Key Features

  • Secure k-anonymity model — password never leaves your machine
  • Real-time breach database lookup via HIBP API
  • Returns count of times the password has been seen in breaches
  • Command-line interface for easy integration
  • Clean, readable Python code with error handling
