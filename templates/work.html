001 — Python Password Checker

Checking passwords against known data breaches using the Have I Been Pwned API — without ever sending your password over the internet

1. What This Does

This is a Python tool that checks passwords against the Have I Been Pwned (HIBP) breach database. If a password has appeared in a known data breach, it tells you how many times — so you know to stop using it.

The key thing is the privacy model. The password is never transmitted anywhere — not even as a full hash. The tool uses k-anonymity to query the API without ever revealing what it is looking for.

The project is structured as three files: a reusable core module, a CLI entry point, and a Tkinter desktop GUI. Everything runs on the Python standard library — no external packages.

Source: github.com/ViliTajnic/passwordchecker

2. The k-Anonymity Model

The HIBP API accepts only the first 5 characters of a SHA-1 hash. It returns all hash suffixes in its database that share that prefix, along with breach counts. The comparison happens locally — the server never sees enough to reconstruct what was queried.

The tool also sends Add-Padding: true in the request headers. This tells the API to include dummy entries in the response, making it harder for a network observer to infer anything from response size alone.

StepWhere it happensWhat is sent / kept
SHA-1 hash the passwordLocalNothing sent yet
Send first 5 hex chars of hash→ HIBP APIe.g. 5BAA6
Receive matching hash suffixes + counts← HIBP APIHundreds of partial hashes (padded)
Check if full hash suffix is in the listLocalPassword never leaves machine

3. passwordchecker_core.py

All the logic lives here: hashing, the API call, response parsing, file reading, and result formatting. Both the CLI and the GUI import from this module — nothing is duplicated between them.

import hashlib
from pathlib import Path
from urllib import error, request


APP_NAME = "Password Checker"
APP_VERSION = "2.1"
API_URL = "https://api.pwnedpasswords.com/range/{}"
REQUEST_HEADERS = {
    "User-Agent": f"passwordchecker/{APP_VERSION}",
    "Add-Padding": "true",
}


def request_api_data(query_char):
    req = request.Request(API_URL.format(query_char), headers=REQUEST_HEADERS)
    try:
        with request.urlopen(req, timeout=10) as response:
            if response.status != 200:
                raise RuntimeError(
                    f"Password API returned status {response.status}. Please try again later."
                )
            return response.read().decode("utf-8")
    except error.HTTPError as exc:
        raise RuntimeError(
            f"Password API returned status {exc.code}. Please try again later."
        ) from exc
    except error.URLError as exc:
        raise RuntimeError(
            "Could not reach the password API. Check your internet connection and try again."
        ) from exc


def get_password_leaks_count(response_text, hash_to_check):
    for line in response_text.splitlines():
        hash_suffix, _, count = line.partition(":")
        if hash_suffix == hash_to_check:
            return int(count)
    return 0


def pwned_api_check(password):
    sha1password = hashlib.sha1(password.encode("utf-8")).hexdigest().upper()
    first5_char, tail = sha1password[:5], sha1password[5:]
    response_text = request_api_data(first5_char)
    return get_password_leaks_count(response_text, tail)


def read_passwords_from_file(file_path):
    path = Path(file_path)
    if not path.is_file():
        raise ValueError(f"Password file not found: {file_path}")

    entries = []
    with path.open("r", encoding="utf-8") as handle:
        for line_number, raw_line in enumerate(handle, start=1):
            password = raw_line.rstrip("\r\n")
            if password:
                entries.append((f"Line {line_number}", password))

    if not entries:
        raise ValueError("The password file does not contain any non-empty lines.")
    return entries


def collect_direct_passwords(passwords):
    return [
        (f"Password {index}", password)
        for index, password in enumerate(passwords, start=1)
    ]


def format_result(label, count):
    if count:
        return (
            f"{label}: found in known breach data {count:,} times. "
            "Do not use this password."
        )
    return (
        f"{label}: not found in the breach corpus checked by this tool. "
        "It may still be weak, so use a long, unique password."
    )


def check_password_entries(entries):
    return [(label, pwned_api_check(password)) for label, password in entries]

request_api_data uses urllib from the standard library — no requests package needed. It catches both HTTPError and URLError separately and raises human-readable RuntimeError messages in either case.

get_password_leaks_count uses str.partition(":") rather than split, which handles the line format cleanly without creating extra list items. It returns int(count) so callers always get a number.

pwned_api_check is where the k-anonymity split happens: SHA-1 hash the password, uppercase it to match the API's format, then separate the first 5 characters (sent to the API) from the tail (kept locally for comparison).

read_passwords_from_file tracks line numbers so each result in a batch run is labelled Line N rather than a generic index. Empty lines are skipped silently.

format_result uses {count:,} for thousands-separator formatting — a breach count of 247116 prints as 247,116.

4. checkmypass.py

The CLI entry point. It handles argument parsing and the three input modes, then delegates everything else to the core module.

#!/usr/bin/env python3
import argparse
import getpass
import sys

from passwordchecker_core import (
    APP_NAME,
    APP_VERSION,
    check_password_entries,
    collect_direct_passwords,
    format_result,
    read_passwords_from_file,
)


def prompt_for_password():
    prompt = "Enter the password to check: "

    if sys.stdin.isatty():
        while True:
            password = getpass.getpass(prompt)
            if password:
                return password
            print("Password cannot be empty. Please try again.")

    password = sys.stdin.readline().rstrip("\r\n")
    if not password:
        raise ValueError("No password was provided on standard input.")
    return password

def parse_args(argv):
    parser = argparse.ArgumentParser(
        description=(
            "Check whether passwords appear in known data breaches using the "
            "Have I Been Pwned range API."
        )
    )
    parser.add_argument(
        "passwords",
        nargs="*",
        help="Passwords to check directly. Avoid this on shared systems because shell history may store them.",
    )
    parser.add_argument(
        "-f",
        "--file",
        help="Check passwords from a text file, one password per line.",
    )
    parser.add_argument(
        "-i",
        "--interactive",
        action="store_true",
        help="Prompt for a password without showing it on screen.",
    )
    parser.add_argument(
        "--version",
        action="version",
        version=f"%(prog)s {APP_VERSION}",
    )
    return parser.parse_args(argv)


def collect_passwords(args):
    if args.file and args.passwords:
        raise ValueError("Use either direct passwords or --file, not both.")
    if args.interactive and (args.file or args.passwords):
        raise ValueError("Use --interactive by itself.")

    if args.file:
        return read_passwords_from_file(args.file)
    if args.passwords:
        return collect_direct_passwords(args.passwords)
    return [("Password 1", prompt_for_password())]


def main(argv):
    try:
        args = parse_args(argv)
        passwords_to_check = collect_passwords(args)
    except ValueError as exc:
        print(f"Input error: {exc}", file=sys.stderr)
        return 2

    print(f"{APP_NAME} v{APP_VERSION}")
    print(f"Checking {len(passwords_to_check)} password(s)...")

    try:
        for label, count in check_password_entries(passwords_to_check):
            print(format_result(label, count))
    except RuntimeError as exc:
        print(f"Error: {exc}", file=sys.stderr)
        return 1

    print("Check completed successfully.")
    return 0


if __name__ == "__main__":
    sys.exit(main(sys.argv[1:]))

The three input modes are mutually exclusive and enforced in collect_passwords. Passing both --file and direct arguments raises a ValueError before any API call is made. Running with no arguments at all falls through to prompt_for_password, which uses getpass when stdin is a TTY (so the password is not echoed) and reads a plain line when stdin is a pipe (useful for scripting).

main returns integer exit codes: 0 for success, 1 if the API call fails, 2 for bad input. These map to standard Unix conventions so the tool composes cleanly with shell scripts and CI pipelines.

5. passwordchecker_gui.py

A Tkinter desktop application that wraps the same core module. It supports both single-password checking and file-based batch checking, with a threaded worker so the UI stays responsive while the API call is in progress.

#!/usr/bin/env python3
import queue
import threading
import tkinter as tk
from pathlib import Path
from tkinter import filedialog, messagebox, ttk
from tkinter.scrolledtext import ScrolledText

from passwordchecker_core import (
    APP_NAME,
    APP_VERSION,
    check_password_entries,
    format_result,
    read_passwords_from_file,
)


class PasswordCheckerApp:
    def __init__(self, root):
        self.root = root
        self.root.title(f"{APP_NAME} Desktop")
        self.root.geometry("720x520")
        self.root.minsize(640, 460)

        self.result_queue = queue.Queue()
        self.worker = None

        self.password_var = tk.StringVar()
        self.file_var = tk.StringVar(value="No file selected")
        self.status_var = tk.StringVar(
            value="Enter one password or choose a file with one password per line."
        )

        self._build_ui()
        self.root.after(150, self._poll_result_queue)

    def _build_ui(self):
        self.root.columnconfigure(0, weight=1)
        self.root.rowconfigure(0, weight=1)

        frame = ttk.Frame(self.root, padding=18)
        frame.grid(sticky="nsew")
        frame.columnconfigure(0, weight=1)
        frame.rowconfigure(5, weight=1)

        ttk.Label(
            frame,
            text=f"{APP_NAME} v{APP_VERSION}",
            font=("Helvetica", 20, "bold"),
        ).grid(row=0, column=0, sticky="w")

        ttk.Label(
            frame,
            text=(
                "Checks passwords against known breach data using the Have I Been "
                "Pwned range API. Your full password is never sent."
            ),
            wraplength=660,
            justify="left",
        ).grid(row=1, column=0, sticky="ew", pady=(8, 18))

        single_frame = ttk.LabelFrame(frame, text="Check one password", padding=14)
        single_frame.grid(row=2, column=0, sticky="ew")
        single_frame.columnconfigure(0, weight=1)

        password_entry = ttk.Entry(single_frame, textvariable=self.password_var, show="*")
        password_entry.grid(row=0, column=0, sticky="ew", padx=(0, 10))
        password_entry.bind("", self._check_single_password)

        ttk.Button(
            single_frame, text="Check Password", command=self._check_single_password
        ).grid(row=0, column=1, sticky="e")

        file_frame = ttk.LabelFrame(frame, text="Check a file", padding=14)
        file_frame.grid(row=3, column=0, sticky="ew", pady=(16, 0))
        file_frame.columnconfigure(0, weight=1)

        ttk.Label(file_frame, textvariable=self.file_var, wraplength=540, justify="left").grid(
            row=0, column=0, sticky="w"
        )

        file_buttons = ttk.Frame(file_frame)
        file_buttons.grid(row=0, column=1, sticky="e", padx=(12, 0))
        ttk.Button(file_buttons, text="Choose File", command=self._choose_file).grid(
            row=0, column=0, padx=(0, 8)
        )
        ttk.Button(file_buttons, text="Check File", command=self._check_file_passwords).grid(
            row=0, column=1
        )

        ttk.Label(frame, textvariable=self.status_var, foreground="#1f4f3f").grid(
            row=4, column=0, sticky="w", pady=(16, 8)
        )

        self.output = ScrolledText(frame, wrap="word", height=14, state="disabled")
        self.output.grid(row=5, column=0, sticky="nsew")

        ttk.Label(
            frame,
            text="Tip: direct command-line passwords are less private than this desktop UI.",
            foreground="#555555",
        ).grid(row=6, column=0, sticky="w", pady=(12, 0))

    def _choose_file(self):
        file_path = filedialog.askopenfilename(
            title="Choose password file",
            filetypes=[("Text files", "*.txt"), ("All files", "*.*")],
        )
        if file_path:
            self.file_var.set(file_path)
            self.status_var.set("File selected. Click 'Check File' to run the scan.")

    def _check_single_password(self, event=None):
        password = self.password_var.get()
        if not password:
            messagebox.showwarning(APP_NAME, "Enter a password first.")
            return
        self._start_background_check([("Password 1", password)], clear_password=True)

    def _check_file_passwords(self):
        file_path = self.file_var.get()
        if file_path == "No file selected":
            messagebox.showwarning(APP_NAME, "Choose a file first.")
            return
        try:
            entries = read_passwords_from_file(file_path)
        except ValueError as exc:
            messagebox.showerror(APP_NAME, str(exc))
            return
        self._start_background_check(entries, source=Path(file_path).name)

    def _start_background_check(self, entries, clear_password=False, source=None):
        if self.worker and self.worker.is_alive():
            messagebox.showinfo(APP_NAME, "A check is already running.")
            return
        if clear_password:
            self.password_var.set("")
        self._set_output("")
        count = len(entries)
        target = source or "interactive input"
        self.status_var.set(f"Checking {count} password(s) from {target}...")
        self.worker = threading.Thread(target=self._run_check, args=(entries,), daemon=True)
        self.worker.start()

    def _run_check(self, entries):
        try:
            results = check_password_entries(entries)
            lines = [f"{APP_NAME} v{APP_VERSION}", ""]
            for label, count in results:
                lines.append(format_result(label, count))
            lines.append("")
            lines.append("Check completed successfully.")
            self.result_queue.put(("success", "\n".join(lines)))
        except RuntimeError as exc:
            self.result_queue.put(("error", str(exc)))

    def _poll_result_queue(self):
        try:
            status, payload = self.result_queue.get_nowait()
        except queue.Empty:
            self.root.after(150, self._poll_result_queue)
            return
        if status == "success":
            self._set_output(payload)
            self.status_var.set("Finished.")
        else:
            self._set_output(f"Error: {payload}")
            self.status_var.set("Check failed.")
            messagebox.showerror(APP_NAME, payload)
        self.root.after(150, self._poll_result_queue)

    def _set_output(self, text):
        self.output.configure(state="normal")
        self.output.delete("1.0", tk.END)
        self.output.insert("1.0", text)
        self.output.configure(state="disabled")


def main():
    root = tk.Tk()
    PasswordCheckerApp(root)
    root.mainloop()


if __name__ == "__main__":
    main()

The threading model is worth noting. _run_check runs in a daemon thread — it calls check_password_entries, which makes the blocking HTTP request, and puts the result into a queue.Queue. The main thread never touches the queue directly; instead, _poll_result_queue is scheduled via root.after(150, ...), which re-queues itself every 150ms. This is the standard Tkinter pattern for non-blocking background work: the event loop keeps running, the UI stays responsive, and the worker communicates back through the queue rather than touching any widgets directly.

The password entry field uses show="*" so the password is masked. After a single-password check, clear_password=True wipes the field so the password does not sit in the UI. The output widget is a ScrolledText kept in state="disabled" except during the brief moment it is being written to, preventing the user from accidentally editing results.

6. Installation and Usage

No packages to install. Clone the repository and run:

git clone https://github.com/ViliTajnic/passwordchecker.git
cd passwordchecker

# Interactive prompt — password not echoed, safest option
python3 checkmypass.py

# Direct arguments — convenient but stays in shell history
python3 checkmypass.py password123 correcthorsebatterystaple

# Batch check from a file, one password per line
python3 checkmypass.py -f passwords.txt

# Desktop GUI
python3 passwordchecker_gui.py

Example CLI output:

Password Checker v2.1
Checking 2 password(s)...
Password 1: found in known breach data 247,116 times. Do not use this password.
Password 2: not found in the breach corpus checked by this tool. It may still be weak, so use a long, unique password.
Check completed successfully.

Pre-built standalone executables for macOS and Windows are generated automatically by GitHub Actions using PyInstaller — no Python installation required to run those.

7. Why This Project

The k-anonymity model is the same one used by browser password managers and enterprise security tools — it is worth understanding from first principles. Beyond the privacy angle, the project is a clean example of how a working script grows into a proper package: the core logic is separated so it can be shared between a CLI and a GUI without any duplication, input handling is made explicit and testable, error messages are user-facing rather than raw exceptions, and exit codes make the tool composable in scripts. All of it without a single external dependency.
