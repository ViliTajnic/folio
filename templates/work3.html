003 — Web Scraper with BeautifulSoup

Web scraping is one of Python's most powerful capabilities for data collection and analysis. This project uses BeautifulSoup and the Requests library to scrape articles from Hacker News — one of the most popular tech news aggregator sites — and returns them sorted by vote count.

The scraper demonstrates core Python skills including HTTP requests, HTML parsing, data extraction, and list manipulation. It's a practical tool for keeping track of trending technology topics and discussions in the developer community.

"I used the example of Hacker News to scrape the Articles with most of the Votes" — https://news.ycombinator.com/news

How It Works

The script sends an HTTP GET request to Hacker News, parses the HTML response using BeautifulSoup, extracts article titles, URLs, and vote counts from the page structure, then sorts and returns the results by vote count in descending order.

Key Features

  • Scrapes Hacker News for top articles by votes
  • Uses BeautifulSoup for reliable HTML parsing
  • Handles pagination across multiple pages
  • Returns sorted results with titles and links
  • Clean code structure with reusable functions
