002 — Flask Presentation Website

Flask is a lightweight WSGI web application framework written in Python. It is designed to be simple and easy to get started with, while being flexible enough to scale up to complex applications. This portfolio website is itself a demonstration of Flask's capabilities.

The application uses Flask's routing system to serve different pages, Jinja2 templating for dynamic HTML rendering, and integrates with a CSV-based database for storing contact form submissions. The entire application runs on PythonAnywhere's free tier, demonstrating Python deployment skills.

"This web application framework is a lightweight wonder and it shows the true power of Python." — xhost.pythonanywhere.com

Technical Stack

Built entirely with Python and Flask, the application features clean URL routing with Flask's @app.route decorator, dynamic page rendering with Jinja2 templates, form handling and data persistence, and static file serving for CSS, JavaScript, and images.

Key Features

  • Python Flask backend with dynamic routing
  • Jinja2 templating engine for HTML generation
  • Contact form with CSV data storage
  • Responsive design with HTML5 UP Editorial template
  • Deployed on PythonAnywhere cloud platform
